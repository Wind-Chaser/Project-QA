# QA - Question Answering
It is Question Answering site. You can ask questions,see answers,give your answers.You can see questions according to categories and subcategories. Admin page also where admin can update or delete any thing.Admin can also add new admin or add new questions or categories, subcategories.It is written entirely in HTML/CSS/JS and PHP with MySQL.

## Home Page
Here, you see all latest questions and there total number of answers given by others.You can see very less part of answer.If you want to see the answer you have to login first.If want to ask question and give answers also you have to login first.
